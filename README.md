# Howard’s Mental Health Portfolio

A personal website built with HTML, CSS, and JavaScript to share my mental health journey, raise awareness, and help family and friends understand how to support me.

## Pages

- `index.html` — welcome page
- `about.html` — about me section
- `support.html` — support guidance
- `guidance.html` — guided support scenarios
- `resources.html` — resources and helpful routines

## Files

- `styles.css` — responsive styling and theme support
- `script.js` — interactive theme toggle, fun fact reveal, and resource filtering

## Run locally

Open `index.html` in your browser or use a static server like Live Server in VS Code.
